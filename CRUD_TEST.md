# CRUD 테스트

## CREATE 게시판

```
{
  METHOD : POST
  url : http://localhost:8000/api/board/
  BODY : {"name":"게시판"}
}

-- 결과 값

{
    "name": "게시판",
    "updated_at": "2023-09-22T08:10:37.000000Z",
    "created_at": "2023-09-22T08:10:37.000000Z",
    "id": 1
}
```

## READ 게시판

-   전체 게시판 READ

```
{
  METHOD : GET
  url : http://localhost:8000/api/board/
}

-- 결과 값

{
    "id": 1,
    "name": "게시판",
    "created_at": "2023-09-22T08:10:37.000000Z",
    "updated_at": "2023-09-22T08:10:37.000000Z",
    "posts":[]
}
```

-   개별 게시판 READ

```
{

}

-- 결과 값

{
    "id": 1,
    "name": "게시판",
    "created_at": "2023-09-22T08:10:37.000000Z",
    "updated_at": "2023-09-22T08:10:37.000000Z",
    "posts":[]
}
```

```
}

-- 결과 값

{
    "id": 1,
    "name": "수정",
    "created_at": "2023-09-22T08:10:37.000000Z",
    "updated_at": "2023-09-22T08:10:37.000000Z",
    "posts":[]
}
```

## DELETE 게시판

```
{
  METHOD : DELETE
  url : http://localhost:8000/api/board/1
}

-- 결과 값

{
    "deleted": true
}
```

## CREATE 포스트

```
{
  METHOD : POST
  url : http://localhost:8000/api/board/1/post
  BODY : {"name":"게시물", "detail":"내용"}
}

-- 결과 값

{
    "name": "게시물",
    "detail": "내용",
    "board_id": 1,
    "updated_at": "2023-09-22T08:21:48.000000Z",
    "created_at": "2023-09-22T08:21:48.000000Z",
    "id": 1
}
```
